fun repeatingTask(delay: Int, period: Int, times: Int = -1, unit: TimeUnit = TimeUnit.TICKS, async: Boolean = false, runnable: BukkitRunnable.() -> Unit): BukkitTask

Schedules a repeating task with the given delay, period, and other optional parameters.

Return

The BukkitTask representing the scheduled repeating task.

Parameters

delay

The delay before the task starts, in the specified time unit.

period

The time between each execution of the task, in the specified time unit.

times

The number of times the task should repeat. Use -1 for indefinitely. Defaults to -1.

unit

The time unit used for the delay and period. Defaults to TimeUnit.TICKS.

async

Specifies whether to run the task asynchronously or synchronously. Defaults to false.

runnable

The code to be executed by the task.

fun repeatingTask(period: Int, unit: TimeUnit, times: Int = -1, async: Boolean, runnable: BukkitRunnable.() -> Unit): BukkitTask

The BukkitTask representing the repeating task.

Parameters

period

The period between each task execution.

times

The number of times the task should be executed. If set to -1, the task will repeat indefinitely until cancelled. Default value is -1.
