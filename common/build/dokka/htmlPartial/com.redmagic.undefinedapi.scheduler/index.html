TimeUnit

enum TimeUnit : Enum<TimeUnit>

Provides an abstraction for representing various time units, including custom support for Minecraft ticks. It allows conversion between standard time units like DAYS, HOURS, MINUTES, SECONDS, and MILLISECONDS using the Java java.util.concurrent.TimeUnit along with custom conversions for Minecraft ticks.
