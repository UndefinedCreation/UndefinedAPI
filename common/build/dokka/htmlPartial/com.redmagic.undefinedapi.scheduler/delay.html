fun delay(ticks: Int, unit: TimeUnit = TimeUnit.TICKS, async: Boolean = false, runnable: BukkitRunnable.() -> Unit): BukkitTask

Delays the execution of a given runnable task for a specified number of ticks.

Return

The BukkitTask representing the delayed task.

Parameters

ticks

The number of ticks to delay the execution of the task.

unit

The time unit to use for the delay (default is TimeUnit.TICKS).

async

Determines whether the task should be executed asynchronously (default is false).

runnable

The runnable task to be executed after the delay.
